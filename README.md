# Analyse CSV - Mini Projet Python

## Description
Ce projet permet d'analyser un fichier CSV contenant des informations sur le temps d'étude et les notes obtenues. Il affiche les statistiques descriptives et un graphique scatter montrant la relation entre le temps d'étude et la note.

## Technologies utilisées
- Python
- pandas
- matplotlib

## Instructions d'exécution
1. Installer les dépendances : `pip install -r requirements.txt`
2. Lancer le script : `python src/main.py`

## Exemple de sortie
- Aperçu des données
- Statistiques descriptives
- Graphique scatter
